GlowLink

ESP32-based automotive LED controller for dual WS2811 underglow strips

ESP32 WS2811 ESP-NOW Automotive OBD-II

Overview

GlowLink is an ESP32-based automotive LED controller designed to manage two WS2811 LED strips for underglow lighting. The system receives vehicle data wirelessly via ESP-NOW from a companion OBD-II bridge module, enabling dynamic LED effects that respond to real-time vehicle data such as speed, RPM, and throttle position.

Hardware

  • ESP32 microcontroller
  • Dual WS2811 LED strip outputs
  • 12V automotive power input
  • ESP-NOW wireless receiver

Features

  • Dynamic underglow effects
  • Wireless vehicle data integration
  • Left/right independent control
  • Real-time response to driving conditions
